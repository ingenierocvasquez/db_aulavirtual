Descripción: El siguiente archivo es una breve guia de la estrictura de la base de datos

Usted puede acceder al esquema completo de 1.9 y 1.10 aquí: ​
En el esquema, el verde representa los datos clave relacionados a usuarios;
el azul representa los datos clave relacionados a cursos y el amarillo representa los datos clave de las sesiones.
Todas las tablas blancas vinculadas a la tabla naranja (la tabla c_item_property) 
son recursos que residen dentro de un curso (foro, documentos, tareas, ejercicios, etc.).
Esta resolución del diagrama sólo puede darle una vista del alto nivel de su escala. 
PAra un diagrama a escala completa, revise  (un archivo PNG de alrededor de 3MB). 
Nosotros usamos el software libre "Dia" para crear el diagrama, así que si desea usarlo y hacer ajustes.
